# Privacy and Compliance

## Overview

This document describes the datasets used in this project and provides recommendations for responsible use in compliance with privacy regulations and ethical guidelines.

## Datasets

This repository includes the following datasets:

1. **CSV and XLSX files** - Thai language datasets containing information relevant to social analysis research
2. **PDF documents** - Research documents in Thai language
3. **JSON data files** - Website configurations and detection patterns in the `data/` directory

## Data Usage

### Owner Statement

The repository owner has confirmed that the data included in this project is permitted for research purposes. However, users should be aware of the following:

### Recommendations

1. **Research Use Only**: The datasets should be used exclusively for legitimate research, educational, or analytical purposes.

2. **Privacy Considerations**: 
   - Do not use this data to harass, stalk, or harm individuals
   - Respect privacy rights and local regulations
   - Consider anonymizing or aggregating data when possible

3. **Legal Compliance**:
   - Ensure your use complies with applicable laws in your jurisdiction
   - Be aware of GDPR (EU), CCPA (California), and other privacy regulations
   - Obtain necessary permissions before using data for commercial purposes

4. **Ethical Guidelines**:
   - Use data responsibly and ethically
   - Consider potential impacts on individuals and communities
   - Follow academic and professional ethical standards

5. **Data Protection**:
   - Do not redistribute sensitive personal information
   - Implement appropriate security measures
   - Delete data when no longer needed

## Social Media Analysis

This tool analyzes public social media profiles. Users should:

- Only analyze publicly available information
- Respect platform terms of service
- Not use the tool for illegal purposes
- Consider rate limiting to avoid overwhelming target services

## Compliance Checklist

Before using this tool or data:

- [ ] Confirm your use case is legitimate and ethical
- [ ] Review relevant privacy laws in your jurisdiction
- [ ] Obtain necessary permissions or consents
- [ ] Implement appropriate data protection measures
- [ ] Document your data processing activities
- [ ] Establish data retention and deletion policies

## Questions or Concerns

If you have questions about data usage, privacy, or compliance, please:

1. Review this document carefully
2. Consult with legal counsel if needed
3. Contact the repository owner through GitHub issues for clarification

## Updates

This document may be updated as regulations and best practices evolve. Users are responsible for staying current with applicable laws and guidelines.
